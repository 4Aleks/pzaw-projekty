# konfiguracja i uruchominie
Rozpakuj projekt i przejdź do folderu projekt04
aby uruchomić wpisz
npm install;
npm install express
npm install morgan
npm install argon2 
npm install express-session
npm install cookie-parser
# włącz komendą npm run dev

# uzytkowanie
 
Rejestracja i Logowanie z użyciem loginu i hasla

można przegladac artykuly i komentarze dodane do nich

gdy jest się zalogowanymn można pisac komentarze i edytowac swoje

jako admin moża dodawac i edytowac artykul i komentarze wszystkich uzytkownikow

za pomoca przycisku motyw można zminic wyglad stronyna cimny i jasny
